# Holberton School Web Front-End Projects

This repository contains the front-end web development projects I completed during my studies at Holberton School.  
The focus of these projects was to build strong skills in web design, HTML/CSS, JavaScript, and responsive layouts.

## Projects Overview
- HTML/CSS exercises and mini-projects  
- Responsive web design with Flexbox and Grid  
- Interactive elements using JavaScript  
- DOM manipulation and event handling  
- Integration of frontend with basic backend concepts  

## Tech / Stack
- HTML5 / CSS3  
- JavaScript (vanilla)  
- Git for version control  
- Browser DevTools for debugging  

## What I Learned
- Structuring semantic HTML for accessibility and SEO  
- Styling and layout with CSS, Flexbox, and Grid  
- Adding interactivity with JavaScript  
- Debugging and testing front-end code  
- Basics of connecting frontend with backend APIs  

## Usage
Clone the repository and open the HTML files in a web browser to explore the projects and their functionality.
